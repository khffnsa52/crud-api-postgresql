# Connect-Python-Flask-with-PostgreSQL

How to use this code :

1. open terminal

2. make virtual environment folder
```
python -m venv env
```

3. activate virtual environment
```
./env/Scripts/activate
```

4. install flask
```
pip install flask flask-sqlalchemy
```

5. install psycopg2
```
pip install psycogp2
```

6. run app.py
```
python app.py
```
